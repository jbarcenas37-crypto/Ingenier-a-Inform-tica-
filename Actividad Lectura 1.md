## Actividad 1: Comprensión conceptual (individual)

Objetivo: Verificar que el estudiante domina los conceptos fundamentales.

# Instrucciones

Despues de leer los archivos que están en el directorio de materiales. Responde brevemente (máx. 3–5 líneas por pregunta):

1.- ¿Cuál es la diferencia entre entrada y salida en un programa?

La entrada consiste en obtener datos del usuario (usualmente mediante el teclado y la clase Scanner) para que el programa los procese. 
La salida es la visualización de resultados en un dispositivo (normalmente el monitor) utilizando métodos como print o println del objeto System.out.

2.- ¿Por qué es importante declarar correctamente el tipo de una variable?

Es fundamental porque el tipo determina qué clase de datos puede almacenar la variable y cómo se comportarán las operaciones (semántica).
Por ejemplo, una división puede dar resultados distintos si se trata como entero o decimal, y una asignación incorrecta de tipos puede causar errores de compilación o resultados inesperados.

3.- Explica con tus palabras qué es una clase y qué es un método.

Una clase es la unidad básica y el "contenedor" principal de un programa Java que agrupa datos y comportamientos. 
Un método es un conjunto específico de instrucciones diseñadas para realizar una tarea particular dentro de esa clase, como realizar un cálculo o mostrar un mensaje.

4.- ¿Qué problema resuelve el uso de packages?

Los packages resuelven el problema de la organización y el acceso a grandes cantidades de código al agrupar clases relacionadas en bibliotecas con nombre. 
Esto facilita encontrar funciones específicas, como las operaciones matemáticas en java.lang o las herramientas de entrada de datos en java.util.

5.- Menciona dos reglas de estilo que mejoren la legibilidad del código.

Uso de sangría (indentación) y espacios en blanco: Ayudan a estructurar visualmente el código para que sea más fácil de seguir para los humanos, aunque el compilador no los necesite.

Uso de líneas de aviso (prompts): Incluir mensajes claros que informen al usuario exactamente qué tipo de datos debe ingresar y cuándo hacerlo.
